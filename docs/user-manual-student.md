# LingoLeap 學生使用手冊
**版本：Alpha 0.2.5 ｜ 適用對象：三年級以上學生**

---

## 開始使用

打開瀏覽器（請使用 **Chrome**），前往：
**https://lingoleap-alpha.web.app**

> 語音功能需要 Chrome 瀏覽器，Safari / Firefox 不支援。

---

## 第一步：選課文

進入網站後，你會看到「故事圖書館」，裡面有幾篇課文。

1. 點擊任一課文卡片
2. 你就會進入那篇課文的學習流程

---

## 學習流程（步驟列）

畫面上方有步驟導覽列，目前 Alpha 開放的步驟：

| 步驟 | 名稱 | 說明 |
|------|------|------|
| 1 | 簡介 | 聽系統介紹作者和故事背景 |
| 2 | 逐段朗讀 | 跟著麥克風唸課文，AI 即時評分 |
| 4 | 生字練習 | 練習寫你唸錯的字 |

---

## 步驟二：逐段朗讀（最主要的功能）

這是最重要的步驟，畫面分成左右兩個區域：

**左側：課文區**
- 目前要唸的段落會**放大並發光**，其他段落會變暗
- 右上角的「注音 ON/OFF」可以開啟或關閉注音符號

**右側：即時回饋區**
- AI 助教會在這裡給你回饋
- 你說的話也會即時顯示出來

### 怎麼朗讀一個段落？

1. **（可選）點「系統朗讀」** — 先聽一遍正確的發音
2. **點「開始朗讀」** — 麥克風啟動，右側會出現「準備好了，請開始朗讀！」
3. **對著麥克風唸出那一段課文**，右側會即時顯示你說的話
4. **唸完後點「完成這段」** — AI 評分，並給你回饋

### AI 怎麼評分？

| 評分 | 結果 | 說明 |
|------|------|------|
| 80% 以上 | 自動進入下一段 | 系統說「唸得很棒！」 |
| 60–79% | 自動進入下一段 | 系統說「唸得不錯！」 |
| 60% 以下 | 請你再試一次 | 系統說「再試一次，你可以的！」 |

> 連續唸對三段以上，會有特別鼓勵喔！

### 其他按鈕

| 按鈕 | 功能 |
|------|------|
| 系統朗讀 | 播放 AI 示範朗讀 |
| 暫停 / 繼續 | 暫停或繼續系統朗讀 |
| 停止 | 停止系統朗讀 |
| 完成這段 | 提交這段的朗讀給 AI 評分 |
| 上一段 / 下一段 | 手動換段（不計分） |
| 停止朗讀 | 停止錄音 |

---

## 完成後：成績報告

所有段落都唸完後，會出現**成績報告**，顯示：
- **字詞準確度（%）** — 你說對的比例
- **朗讀速度（字/分鐘）**
- **朗讀分析** — 簡短說明你這次的表現

點「回圖書館」可以回去選另一篇課文。

## 常見問題

**Q：麥克風沒有反應？**
請確認瀏覽器已允許麥克風權限。點選網址列左側的鎖頭圖示 → 麥克風 → 允許。

**Q：系統聽不懂我說的話？**
說話時請靠近麥克風，語速放慢，字字清楚。周圍環境盡量安靜。

**Q：注音符號沒有出現？**
等待 1–2 秒讓資料載入，或重新整理頁面。

**Q：只能用 Chrome 嗎？**
是的，語音辨識功能目前只支援 Chrome 瀏覽器。
