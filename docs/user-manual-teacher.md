# LingoLeap 老師使用手冊
**版本：Alpha 0.2.5 ｜ 適用對象：帶班老師**

---

## Alpha 版說明

目前是 **Alpha 測試版**，主要功能是讓學生體驗逐段朗讀流程。
老師的管理後台（帳號登入、班級管理、學生進度查看）將在後續版本推出。

**Alpha 期間，老師的主要工作是：**
1. 請學生用自己的裝置開啟網站
2. 引導學生進行朗讀練習
3. 觀察學生操作並收集回饋

---

## 給學生的操作環境要求

請在讓學生使用前確認以下事項：

| 項目 | 要求 |
|------|------|
| 瀏覽器 | **Chrome**（語音辨識必須） |
| 裝置 | 電腦、平板（建議）或手機 |
| 麥克風 | 內建或外接麥克風均可 |
| 網路 | 穩定的 Wi-Fi |
| 網址 | **https://lingoleap-alpha.web.app** |

> iPhone 的「接力」功能可能將麥克風導向 Mac，請參考 `docs/Stop iPhone as a microphone.md`。

## 學生學習流程（老師需了解）

學生選完課文後，依序進行：

**步驟 1 — 簡介**
系統朗讀作者介紹與故事背景，幫助學生進入情境。

**步驟 2 — 逐段朗讀**（核心功能）
- 課文分段顯示，每次只看一段
- 學生可先點「系統朗讀」聽示範
- 再點「開始朗讀」唸給麥克風聽
- 唸完點「完成這段」提交
- AI 評分並給鼓勵語句
- 準確度 ≥ 60% 自動進下一段；< 60% 請學生再試

**步驟 4 — 生字練習**
唸錯的字自動被選出，學生透過筆順動畫練習書寫。

**完成後**：顯示準確度與朗讀速度的成績報告。

---

## 建議課堂引導方式

1. **課前**：先在投影機上示範一次完整流程，讓學生知道每個按鈕的功能。
2. **練習前**：提醒學生說話速度放慢、字字清楚，麥克風收音才準確。
3. **遇到問題**：若系統一直評為「再試一次」，可引導學生先點「系統朗讀」聽正確發音再試。
4. **課後**：請學生描述哪些段落最難，收集對課文難度與 AI 回饋語氣的意見。

## Alpha 期間希望收集的回饋

請協助記錄以下項目，回報給開發團隊：

- [ ] AI 評分是否符合實際朗讀水準？（過嚴 / 過鬆）
- [ ] 鼓勵語句的語氣是否適合學生年齡？
- [ ] 注音顯示是否有錯誤？
- [ ] 哪些按鈕讓學生感到困惑？
- [ ] 學生最常卡在哪個步驟？
